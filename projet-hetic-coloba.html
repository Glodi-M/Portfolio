HETIC COLLAB

Cliquer ici pour acceder à l'application →

Bonjour,
je vous présente le projet que nous avions réalisé dans le cadre du cours de No Code / Low code.
Il s'agit d'un projet d'application web intitulé HETIC COLLAB. L'objectif principal de ce projet était de réaliser une application qui permet aux étudiants la facilitation de collaborer pour la réalisation des projets et de permettre à l’école de les suivres..


Ce projet comprend plusieurs fonctionnalités telles que :

  • S'inscrire
  • Modifier son profil
  • Recherche de groupe
  • intégrer le groupe
  • collaborer avec le membre du groupe
  • Ajouter les liens
  • Visionner les membres du groupe

Les outils et technologies utilisés :


Nous avons utilisé:

un outil open source permettant de créer des dessins, des schémas et des diagrammes en ligne avec un style visuel manuel

une plateforme de développement d'applications no-code qui permet de créer facilement des applications mobiles ou web à partir de Google Sheets ou d'autres sources de données, sans nécessiter de compétences en programmation.



Ce projet est entièrement responsive/fonctionnel/optimisé pour les navigateurs modernes.


Ce projet m'a permis de développer plusieurs compétences importantes, notamment dans le No Code avec l'usage de l'outil Glide, excalidraw et travail en équupe.



Je suis très satisfait du travail accompli et des connaissances acquises grâce à ce projet.

À l'avenir, j'aimerais améliorer ce projet en ajoutant des nouvelles fonctionnalités, optimiser le code, ou intégrer de nouvelles technologies.


Merci pour votre attention.
