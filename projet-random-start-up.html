Bonjour,

je vous présente le projet que nous avions réalisé dans le cadre du cours de HTML/CSS.
Il s'agit d'une réproduction et d'une intégration d'une maquette du site Random Start-up. L'objectif principal de ce projet était de réproduire la maquette en HTML/CSS telle est définie sur le livravable Figma.

Ce projet est entièrement responsive/fonctionnel/optimisé pour les navigateurs modernes.


Ce projet m'a permis de développer plusieurs compétences importantes, notamment en langage HTML et CSS, comment structurer la page et mettre le style aussi rendre la page responsive.

À l'avenir, j'aimerais améliorer ce projet en ajoutant des nouvelles fonctionnalités, optimiser le code, ou intégrer de nouvelles technologies.
